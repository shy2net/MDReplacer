MDReplacer is a simple utility aimed for Windows 10 (and above+). It allows moving between desktops easily using the Shift + Mouse wheel up\down.
Make sure you have multiple desktops before trying to move between them. You can create a new desktop by clicking on Ctrl + Win + D.
